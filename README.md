# T20-Cricket-wordcup-2022-scorecard-analysis-using-python-and-powerbi
Perfomed Data Analytics on T20 Cricket Wordcup 2022 data to build a team of Perfect 11 players.
Used web scrapping to scrap the data from espncricinfo website.
used Python,Jupyter notebook and pandas for data cleaning and transforming.
used matplotlib for data visualization
Built an interactive dashboard using powerbi to generate insights of best playing 11.
In powerbi used power query for further data transforming and DAX for parameter building
